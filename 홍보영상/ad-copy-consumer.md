> 10MWT + TUG 통합 시스템 기준. 내부 기술 스택 제외, 소비자가 체감할 수 있는 가치 중심 구성.
> 영상 제작·SNS·랜딩페이지·소개자료 등 다목적 활용.

## 1. 오프닝 — 문제 제기

### Hook A — 가격 충격
```
보행분석 장비, 아직도
수천만 원짜리 매트 위에서 하시나요?
```

### Hook B — 현실 자각
```
눈으로 보는 보행분석,
얼마나 정확하다고 확신하십니까?
```

### Hook C — 숫자 임팩트
```
보행속도 0.1m/s 차이가
낙상 위험 1.5배를 가릅니다.

— Verghese et al., 2009
```

### Hook D — 평가의 한계
```
스톱워치 하나로
기립, 보행, 회전, 착석을
모두 평가할 수 있습니까?
```

### Hook E — 숨겨진 정보
```
걸음걸이 안에는
속도보다 중요한 정보가 있습니다.

기립 속도, 회전 안정성, 좌우 대칭,
어깨와 골반의 기울기까지.
```

## 2. 제품 소개 — 전환

### 한 줄 소개
```
스마트폰 하나로 시작하는
임상급 AI 보행분석
```

### 핵심 메시지
```
10미터를 걸으세요.
의자에서 일어나 걸으세요.

AI가 당신의 보행을 읽어드립니다.
```

### 방법 강조
```
촬영하세요.
특별한 장비는 필요 없습니다.

마커 없이,
센서 없이,
고가의 매트 없이.

환자의 키 정보 하나면 충분합니다.
```

### 두 가지 검사, 하나의 시스템
```
10MWT — 10미터 보행검사
TUG  — Timed Up and Go

두 가지 표준 임상 검사를
하나의 시스템에서 수행합니다.
```

## 3. 10MWT 측정 항목

### 기본 측정
```
보행속도 (m/s)
보행시간 (초)
```

### 질환별 확장 변수
```
분속수 (Cadence)
좌·우 보폭 시간 (Step Time)
좌·우 보폭 시간 비대칭
좌·우 활보장 시간 (Stride Time)
이중 지지기 비율
보행 규칙성
어깨·골반 기울기 추이
```

### 기능 요약 — 10MWT
```
보행속도·시간 자동 측정
질환에 따른 확장 변수 자동 활성화
좌우 대칭성 분석
낙상 위험 점수 (0–100점)
연령·성별 기준 정상치 비교
치료 전후 객관적 변화 추적
```

## 4. TUG 측정 항목

### 5단계 자동 감지
```
1단계 — 일어서기 (Stand Up)
2단계 — 걸어가기 (Walk Out)
3단계 — 돌아서기 (Turn)
4단계 — 걸어오기 (Walk Back)
5단계 — 앉기 (Sit Down)

각 단계의 소요시간을 자동으로 분리합니다.
```

### 기립·착석 분석
```
기립 속도
착석 속도
손 지지 사용 여부
동작 부드러움 점수
반응 시간
첫걸음까지의 시간
```

### 정면 영상 분석 (듀얼 카메라)
```
어깨 기울기 (평균·최대)
골반 기울기 (평균·최대)
측방 흔들림 (체중이동 분석)
압력중심 궤적
기울기 이상치 자동 감지
```

### 질환별 확장 변수
```
팔 흔들기 속도·비대칭 (파킨슨)
회전 최대 각속도 (파킨슨·뇌졸중)
체간 각속도 (낙상 위험)
좌우 보폭 비대칭 (뇌졸중)
관절 가동범위 (슬관절·고관절 OA)
기립·착석 부드러움 (Jerk 분석)
분당 걸음수 (Cadence)
발 높이 (Foot Clearance)
```

### 기능 요약 — TUG
```
5단계 자동 감지 및 시간 분리
측면 + 정면 듀얼 영상 분석
기립·착석 세부 분석
체중이동·자세 안정성 분석
질환에 따른 확장 변수 자동 활성화
단계별 전환 시점 프레임 캡처
```

## 5. 정확도 — 신뢰할 수 있는 측정

### 10MWT 정확도
```
실측 시간과의 평균 오차: 0.015초

7.33초 → AI 측정 7.34초 (오차 0.01초)
7.58초 → AI 측정 7.60초 (오차 0.02초)
9.33초 → AI 측정 9.30초 (오차 0.03초)
10.43초 → AI 측정 10.43초 (오차 0.00초)

4개 검증 영상 모두
신뢰도 100/100 달성
```

### 교차검증
```
Leave-One-Out 교차검증 결과
평균 오차 0.018초

과적합이 아닌,
검증된 정확도입니다.
```

### TUG 정확도
```
4세트 듀얼 영상 분석
전체 세트 신뢰도 100/100 달성
5단계 구간 감지 정상 작동
정면 영상 기울기 분석 정상 작동
```

### 한 줄 요약
```
스톱워치와 0.015초 차이.
그런데 14개 변수를 더 측정합니다.
```

## 6. 질환별 맞춤 분석 — 9가지 프로파일

```
환자 진단명을 입력하면
AI가 분석 방식을 자동 조정합니다.
```

### 지원 질환
```
파킨슨병
뇌졸중 (편마비)
다발성 경화증
척수 손상
뇌성마비
슬관절 OA / 인공슬관절
고관절 OA / 인공고관절
낙상 위험
일반 (기본)
```

### 질환별 핵심 분석
```
파킨슨병
→ 보행 리듬 불규칙, 팔 흔들기 비대칭,
  회전 각속도 저하, Shuffling 감지

뇌졸중
→ 좌우 보폭 비대칭(20-40%),
  이중 지지기 증가, 편측 팔 흔들기 저하

슬관절·고관절 OA
→ 관절 가동범위 측정,
  통증 회피 보행 패턴 감지

낙상 위험
→ 보행 변동성 증가 감지,
  이중 지지기 증가, 체간 불안정
```

## 7. 비교 — 경쟁 우위

### 장비 비교
```
GAITRite 전자 매트        ₩15,000,000~
관성 센서 (IMU) 시스템     ₩5,000,000~

AI 보행분석 시스템
스마트폰 1대
```

### 기존 방식과 비교
```
[기존 10MWT]
스톱워치 + 눈 측정
→ 보행속도 1개만 측정

[AI 10MWT]
영상 1개 업로드
→ 보행속도 + 질환별 확장 변수
  + 판정 + 대칭성 + 낙상 위험 점수
```

```
[기존 TUG]
스톱워치로 총 시간만 측정
→ "14.5초, 양호"

[AI TUG]
측면 + 정면 영상 업로드
→ 5단계 시간 분리
  + 기립·착석 속도
  + 손 지지 감지
  + 어깨·골반 기울기
  + 체중이동 분석
  + 질환별 확장 변수
```

### 핵심 한 줄
```
측정의 객관성을 높이고
장비의 문턱을 낮췄습니다.
```

### 보행 능력 등급
```
Perry et al. (1995) 기준
4단계 보행 능력 자동 분류

Lv.1 정상 보행     — ≥ 1.2 m/s  독립적 지역사회 활동
Lv.2 지역사회 보행  — 0.8~1.2    일부 제한
Lv.3 제한적 보행    — 0.4~0.8    보조 필요
Lv.4 실내 보행     — < 0.4      실내만 가능
```

### 보행 경로 분석
```
환자의 출발지와 도착지를 입력하면
지도 위에 경로를 표시하고
환자의 보행속도에 따른 예상 소요시간을 계산합니다.

"집에서 병원까지 얼마나 걸릴까요?"

일반인 기준 소요시간과 함께 비교합니다.
```

### 맞춤 목표 설정
```
편의점까지 50m, 공원까지 200m,
횡단보도 30m...

치료사가 환자 주변 목적지를 설정하면
보행속도를 기반으로 예상 소요시간을 계산합니다.

횡단보도는 신호시간과 비교하여
안전하게 건널 수 있는지 확인합니다.
```

### 재활 권장 시스템
```
낙상 위험 등급과 진단명에 따라
맞춤 재활 프로그램을 자동으로 권장합니다.

파킨슨병 → 리듬 청각 자극, 이중과제 훈련
뇌졸중  → 체중이동 훈련, 마비측 강화
OA     → 관절 ROM 운동, 수중 치료

위험 등급별 재평가 주기도 안내합니다.
```

## 9. 낙상 위험 평가

### 점수 체계
```
보행속도와 보행시간을 종합하여
0–100점 낙상 위험 점수를 산출합니다.

90–100점  정상      낙상 위험 낮음
70–89점   경도 위험  모니터링 권장
50–69점   중등도 위험 적극적 개입 필요
0–49점    고위험     즉각적 개입 필요
```

### TUG 평가 기준
```
< 10초    정상   독립적 이동 가능
10–20초   양호   대부분 독립적
20–30초   주의   보조 기구 필요 가능
≥ 30초    위험   낙상 위험 높음
```

### 연령·성별 기준 비교
```
환자의 보행속도를
동일 연령·성별 정상치와 비교합니다.

20대부터 80대까지
남성·여성 각각의 기준 데이터를 보유하고 있습니다.

— Bohannon, 2011
```
